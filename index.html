Powered by Node.JS

The entire SchoolTrackr platform has been built from the ground up to be as fast as possible, and Node helps to make it a reality. It also gives us the ability to support realtime events across the platform.

Zero Device Lock-In

Any device with internet access can use the platform, regardless whether it's Apple, Android, or Windows. The web based client ensures cross device compatibility

Simple REST API

We provide an easy to use REST API for use in other applications, so that you can easily extend the functionality of the platform, or use it in other services.
